# 🌍 TerraGuard: Landslide Prediction Model

TerraGuard is an intelligent landslide prediction system that combines meteorological forecasting, machine learning models, and geospatial data to provide early warnings for landslide-prone regions in India. This system aims to enhance disaster preparedness and protect vulnerable communities.

## 🚀 Features

- ⏳ **30-Day Weather Forecasting** using time series models.
- 🌧️ **Rainfall & Soil Saturation Analysis** from NASA POWER datasets.
- 🌲 **Landslide Hazard Assessment** using LHASA (Landslide Hazard Assessment for Situational Awareness).
- 🌐 **City-Wise Prediction Dashboard** with maps and clickable safe zone links.
- 📶 **Offline Accessibility** for use in low-connectivity areas.
- 🧠 **Machine Learning Integration** using Random Forest Regressor for landslide probability prediction.
- 📱 **Responsive Frontend Application** with intuitive UI for public access and awareness.
